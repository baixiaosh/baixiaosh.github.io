为什么不是我，而是别人

github weibo
2015年7月20日

2015，两年了。

2015，两年了。

这算一篇总结吧，工作两年的总结。

2015，夏。天气异常的热，偶尔下场雨，都以为是在下冰雹。我们在这怪异的天气下默默的做着政府的“奴隶”。在浑噩的时光里，我甘愿作一条水草。

那年，学校的我们多么天真可爱，从未感到今日的孤独。每天上课，每天抢饭，每天回寝室打游戏。每天做着同样的事情，却不感到无聊。如今回想起来，感觉好像是童年。是青春，青春易逝，我们没有牢牢地抓住，已经被时间狠狠地拽走。在寻找青春的那些年，丢掉了多少青春都没有找到，走过的痕迹，都只是回忆。

而现在，每个人都自己努力的拼搏着，为了将来美好的生活，自己大部分的时间都放在了工作上。这是必然的。淡了，淡了。淡的不仅是欢笑，淡的不仅是那份追求青春的执着，淡的是彼此之间的联系。你走了，我不送；你来了，我迎接。你来与不来，我都做这自己的事。你在路上，是你的事；你在我家门口，才是我的事。

QQ是个好的应用软件，我们是好友，你不删我，我不删你，我们永远是好友。而彼此之间的聊天记录，半年？一年？两年？呵呵的笑吧。

时间是岁月的流痕，你擦不掉的。

2013，事业懵懂的初期。总结，各种骗与被骗。至于骗与被骗的过程，在此略过。在中国，我觉得都会这样吧，都是给政府打工。而为了自己的一点利益，就去骗他人，是通病。

2014，事业懵懂的中期。总结，各种换。在骗与被骗之后，选择不停的还工作。自己能力不够，被公司挑。够了，自己挑公司。做过美工，java，前端。最后选择前端，她能激发我的兴趣，如果没了兴趣，你工作是注定枯燥无聊的。你愿或者不愿，他也是枯燥的。热爱她，她会给你回报的。

2015，事业，什么鬼？什么事业啊，就是为了混口饭吃，能吃饱就好了。学会了装逼，学会了各种装逼。

最近，不知道在瞎忙什么，说好的每天都要记录一下的，现在好久都没写了。还是要继续坚持啊，俗话说：什么什么，贵在坚持。不然什么时候才能变成CEO啊，迎娶白富美，走向人生巅峰啊。我可不想分分钟被拖进优衣库试衣间。

买了相机，说好的装逼。却偏偏喜欢上了摄影。忙碌的我，剩下几百张图片在那里，不经修饰。帅锅，美女，走，约起。我去当电灯泡，给你们照相。

过去的，变成经历，经验，以及回忆。

给我身边的所有人，你总结过了么？没有的话，拿起的手里的笔，或手机，写下那份珍贵的回忆。

2015-7-20，柏小生 笔
