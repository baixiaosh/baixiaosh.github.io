为什么不是我，而是别人

github weibo
2015年4月7日

关于我

柏小生，性别和年龄不祥。前端工程师，目前在一家互联网公司就职。似乎是2014年毕业的专科生？对，就是专科生。

在高考的时候，为了一个妹子少考了300多分的傻逼，然后顺利的进入了阿坝师范高等专科学校（现在叫阿坝学院）。

据说在填志愿的时候，他做了一个测试，测试上显示他将来会从事的职业，上面有IT啊啥的，等等等，然后脑中血液凝固，思维禁止，不对，是时间禁止，选了个计算机软件技术。他也不知道这个计算机是个什么鬼，初中的时候就知道电脑是个玩意儿，但是计算机是个什么鬼，完全不知道啊。像百度，谷歌，麻花腾，麻云，从来没听过，只知道有QQ这个可以在电脑上发消息的东西。

他上了大学之后，知道了啥是计算机，啥是百度，啥是谷歌，计算机能干啥，百度谷歌能干啥，还能用java写个简易QQ。逼格一下上升了好几个档次。他觉得自己很牛逼很牛逼，大学课还没上完，就兴致勃勃的出去找工作了。

第一份工作是啥来着？华润万家做美工。呵呵，如此高大上的公司，如此高大上的职业，这个美工竟然不是现在做网页的那种美工。听他说，这个美工TMD就是超市里面布置气氛的。如果不是国企，如果不是大表哥介绍的，他分分种辞职不干了，还在那里被折磨了那么久。说好的IT呢。说好的IT呢。他想了想，抹了眼泪，对经理说：“我不干了！”经理说：“你为啥啊？”他45°望着天花板，他说他要做IT，他要做IT。就这样他背着行囊，做IT去了。

做IT做啥呢？他大学主要课程是java，他索性找了份java的工作先做着。做着做着就宕机了。他开始丢失在IT这个行业里了，这是他想要的工作吗？天天加班，两点一线。生活的意义哪儿去了？不喜欢自己的职业，不是一个好工程师，也做不出好的产品。

他不断的在IT业界摸索，去寻找属于自己的风口。俗话说：站在台风口，是猪也能飞起来。

终于有一天，有个人告诉他：来做前端吧，你TM太有潜力。他想了很久很久很久，想想之前找前端工作因没有工作经验，没有项目经验，没有作品，空有一腔报复和理论知识而处处碰壁。你TM说我有潜力？工作都找不到，潜力有毛用。但是这个职业又如此的吸引着他，他又是那么的喜欢这个前端的工作。一次偶然的台风来临，是跳进去还是不跳进去呢？Duang，就这样开始了他的前端工程师的职业道路之旅。

他特别喜欢前端，喜欢研究前端的东西。他还需要一个大的平台去拓展他的视野，去运用他所学的知识。他曾说不要问他为什么喜欢前端，喜欢仅仅是一种表达我对前端的态度。

他不再觉得自己是有多么的牛逼，因为在各个行业都有那么多大神牛逼的存在；
他有很多不足，只有通过不断的努力学习，从IT业界汲取营养，才能长成苍天大树。

他面试座右铭：为什么不是我，而是别人。
